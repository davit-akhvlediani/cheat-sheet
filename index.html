CSS Selectors

Pattern Name Description
* Universal selector Matches any element
X Type Selector Matches any X element (i.e. an element of type X)
.X Class Selector Selects element with id="X"
X.Y Chaining Selector Select X elements with the Y class
#X ID Selector Selects all elements with ID="X"
X Y Descendant Selector Matches any Y element that is a descendant of an X element
X > Y Child Selector Matches any Y element that is a child of an X element
X:first-child The :first-child pseudo-class Matches element X when X is the first child of its parent
X:link
X:visited		 The link pseudo-classes Matches element X if X is the source anchor of a hyperlink of which the target is not yet visited (:link) or already visited (:visited)
X:active
X:hover
X:focus		 The dynamic pseudo-classes Matches X during certain user actions
X:lang(A) The :lang() pseudo-class Matches element of type X if it is in (human) language A (the document language specifies how language is determined)
X + Y Adjacent selectors Matches any Y element immediately preceded by a sibling element X
